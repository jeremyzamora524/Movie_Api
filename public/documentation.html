You will be able to sign up, update your personal information and make a list of your favorite movies.

endpoints and descriptions


Method EndPoint Response
GET /movies JSON of all movies
GET /movies/[title] JSON containing details about movie by title
GET /movies/genres/[title] Details about Genre of movie by title
GET /directors/[name] Details about director by name
POST /users Allows user to register
PUT /users/[name] Allows user to update their information
PUT /favorites/[id]/[title] Allows user to add movie to their list of favorites
DELETE /users/[name] Allows user to delete account
